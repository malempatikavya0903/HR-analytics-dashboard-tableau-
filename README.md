# 👥 HR Analytics Dashboard (Tableau)

## 📌 Project Overview

This project is an interactive **HR Analytics Dashboard** developed using **Tableau** to analyze workforce performance, employee attrition, demographics, and job satisfaction. The dashboard provides HR teams with actionable insights to monitor employee trends, improve retention strategies, and support data-driven decision-making.

---

## 🚀 Key Features

- Developed an interactive HR Analytics Dashboard using Tableau.
- Built KPI cards to monitor Employee Count, Active Employees, Attrition Count, Attrition Rate, and Average Age.
- Analyzed employee attrition across different departments and education fields.
- Visualized employee distribution by Age Group and Gender.
- Created Job Satisfaction analysis across different job roles.
- Compared attrition rates across multiple age groups using interactive charts.
- Implemented dynamic filters for interactive workforce analysis.
- Designed an executive dashboard to support HR decision-making.
